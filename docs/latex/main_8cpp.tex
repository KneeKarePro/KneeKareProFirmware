\doxysubsection{Detailed Description}
The goal of this project is to enable our Knee Rehab device to store data on an SD card and when a user connects to the device via Bluetooth, the device will send the data to the user\textquotesingle{}s Laptop. The project will use an ESP32 microcontroller and the Arduino IDE. The project will make use of Free\+RTOS tasks to handle the Bluetooth and SD card functionality concurrently.
